# Backlink & Authority Plan (2026-06-07)

**Why this file exists:** Google Search Console shows 675 pages discovered but only **7 indexed**. The site is technically healthy; the blocker is **domain authority** (it is young and has almost no inbound links, so Google barely crawls it). The fix is real backlinks + time. More pages/photos do nothing until this moves.

**Golden rules**
- Always write the brand as **SuppDoc.io** (the competitor suppdoc.co exists).
- Never paste iHerb/affiliate links off-site. Link to suppdoc.io tool pages (`/quiz`, `/audit`, `/interactions`, `/ingredients`).
- Quality > quantity. 10 real links from real sites beat 200 spam directory links (which can hurt).
- Lead with the **free tools** (quiz, stack audit, interaction checker). Tools earn links; pill catalogs don't.

### 1. Product Hunt launch  ⭐ biggest single win
- One-time traffic spike + a permanent dofollow backlink from a high-authority domain + often picked up by AI tools.
- Position it as a **free tool**: "SuppDoc.io, a free, evidence-based supplement stack builder + interaction checker."
- Effort: medium (1 evening). I can write the full PH listing (tagline, description, first comment, maker comment) for you to paste.
- Launch Tue–Thu, 00:01 PT. Ask 5–10 friends to upvote/comment in the first 2 hours.

### 2. Free, reputable directories (1–2 hrs, easy, do all)
Submit the site (name = SuppDoc.io, same one-line description everywhere for entity consistency):
- **AlternativeTo** (alternativeto.net) — list as alternative to Examine, Fullscript, supstack.
- **SaaSHub** (saashub.com)
- **Crunchbase** (crunchbase.com) — create a free company profile (also helps the AI/Wikidata entity).
- **BetaList** (betalist.com) — for newer products.
- **There's An AI For That / Futurepedia / Toolify** — the site is AI-powered; these are easy dofollow links + AI-tool traffic.
- Skip anything that asks for payment or looks like a link farm.

### 3. HARO-style press links  ⭐ highest authority
Journalists need expert quotes daily. Answering = links from real news/health sites (huge authority).
- Sign up free: **Featured.com** (formerly HARO), **Qwoted**, **SourceBottle**.
- Filter for: supplements, nutrition, vitamins, wellness, longevity.
- Reply as "founder of SuppDoc.io, an evidence-based supplement resource." (A clinical reviewer credential would make these land far more often — worth recruiting.)
- I can draft each response for you in 5 minutes when a relevant query appears.

### 4. Show HN + Indie Hackers (the tools angle)
- **Hacker News → "Show HN: SuppDoc.io, a free supplement interaction checker"** — link the `/audit` or `/interactions` tool. Can send real traffic + a backlink.
- **Indie Hackers** — post a "building in public" milestone, link the tool.
- Effort: low. I'll write both posts.

### 5. Wikidata entity (for Google + AI citations)  ⭐ GEO
- Create a **Wikidata** item for SuppDoc.io (organization, website, founded, description). This is how Google's Knowledge Graph and AI assistants learn you exist as an entity. Pairs with the Organization JSON-LD already on the site.
- Also fill **Organization.sameAs** in the site schema once social profiles exist (currently empty per notes).

- **"Best supplement interaction checker" roundup outreach**: search that phrase, find the listicles, email the authors offering the free tool for inclusion. (I'll write the outreach email.)
- **Competitor alternative pages**: get listed on "Examine alternatives", "Fullscript alternatives", "supstack alternatives" pages.
- **Reddit** (separate track, see week-2-pack.md): slow-burn authority + the #1 AI-citation surface.
- **Niche directories**: nutrition/health tool directories, biohacking resource lists.

## What to AVOID (can hurt the domain)
- Paid "1000 backlinks for $5" gigs (Fiverr link blasts) → toxic, can trigger penalties.
- Blog-comment spam, forum signature spam.
- Private blog networks (PBNs).
- Exact-match anchor-text stuffing. Use natural anchors ("SuppDoc.io", "this free interaction checker").

---

Indexation of a young domain ramps over **weeks to a few months** as authority builds. Expect the indexed count to climb from single digits → dozens → hundreds as these links land and Google raises the crawl budget. There is no instant switch; consistency wins.
